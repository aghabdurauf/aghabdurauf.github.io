**Key Achievements:**
- Developed Spark applications and ETL/ELT pipelines transforming complex JSON patient data into optimized parquet formats

- Reduced manual data retrieval time by 50% and enabled near real-time analytics
- Documented comprehensive data lineage for compliance and audits
